## Setup

1. Clone this repo

```bash
git clone https://github.com/gideon-del/hng-task-3.git
```

2. Go to the cloned directory

```bash
cd ./hng-task-3
```

3. Install dependencies

```bash
pnpm install

```

4. Run the project

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
